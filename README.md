# Assignment SQL: Gestão de Vendas

## Introdução
O objetivo deste exercício é trabalhar com bases de dados em MySQL utilizando consultas SQL.

## Objetivos
1. Crie a base de dados utilizando o arquivo SQL em `resources/database_setup.sql`.
2. Implemente as queries no arquivo `solucao.sql` para responder às seguintes questões:
   - [ ] a) Listar todos os produtos disponíveis.
   - [ ] b) Obter o cliente que realizou mais compras.
   - [ ] c) Listar todas as vendas realizadas dentro de um intervalo de datas.
   - [ ] d) Calcular a receita total de todas as vendas.
   - [ ] e) Contar quantas unidades de cada produto foram vendidas (ordem decrescente).

3. Verifique os resultados localmente na sua instalação de MySQL.
4. Submeta o arquivo `solucao.sql` alterado com as respostas das queries.

## Submissão
- Faça commit do arquivo modificado no repositório Git e realize o  push para o GitHub.
- O sistema do GitHub Actions irá verificar automaticamente o seu trabalho.
